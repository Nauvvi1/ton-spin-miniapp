# TON Spin Mini App

Simple Telegram mini app where you spin a reel and win TON rewards.

# Demo Video

🎥 https://www.youtube.com/watch?v=NPs--krGp-M

------------------------------------------------------------------------

## What is this?

TON Spin is a fast, interactive mini app built for Telegram.

-   Tap **Spin**
-   Watch the reel animate
-   Get a random TON reward

Every spin gives a prize --- sometimes small, sometimes big.

------------------------------------------------------------------------

## How it works

-   Each spin costs **1 TON**
-   Rewards vary:
    -   small: `0.01 TON`
    -   medium: `0.2 – 5 TON`
    -   big: `10 – 50 TON`
-   Results are calculated on the **server**
-   Animation is smooth and client-side

------------------------------------------------------------------------

## Features

-   Smooth reel animation (realistic slowdown)
-   Auto Spin mode
-   Random reward system
-   Telegram Mini App ready
-   Instant spin (no waiting for server)

------------------------------------------------------------------------

## Tech (short)

-   Frontend: React + Vite + TypeScript
-   Backend: Node.js + Express
-   Bot: grammy
-   Transport: Telegram WebApp

------------------------------------------------------------------------
